Forms
 

  1. ប្រភេទរបស់ Form

    Form នៅក្នុង HTML ត្រូវបែងចែកជាពីរគឺ៖

    • Structure or Shell : ជាការកំណត់រចនាសម្ព័ន្ធ និងសំបករបស់ Form ដែលមាន Field ដូចជា Textbox, label, Buntton ដែល​ User មើលឃើញលើនិងបំពេញលើ Web Page។

    • Processing script: ជា server-side scrpit ដែលនឹងយកព៏ត៌មាននៅក្នុង form ទៅបំពេញនៅក្នុង database។

    2. ក្នុង Form យើងអាចបង្កើត form element ដូចជា textbox, radio buttons, checkboxes,dropdown menus, textarea, និង button។ អ្នកត្រូវឱ្យឈ្មោះ Field នីមួយៗដោយ name attribute ដើម្បីសំគាល់ពេលយក Data ទៅ Process។

  2. ទម្រង់ទូទៅរបស់ HTML Form

    Form មានផ្នែកសំខាន់ៗមួយចំនួនដូចជា៖

    • Form tag ដែលត្រូកំណត់នូវ url របស់ sever-script ដែលនឹង process form data form elements និង​ menu

    • Submit button សំរាប់ send ទិន្នន័យទៅកាន់ sever

    ទម្រង់របស់​ Form:

    <Form method="post|get"
    name="form-name"
    action="url">
    -----------------------------
    </Form>

  3. <Input> tags

    បង្កើត Textbox សំរាប់ឱ្យ User បញ្ចូល ទិន្នន័យអាចផ្ទុក text បានមួយបន្ទាត់។

    <input type="text"
    name="txt-name"
    value="defaul-value"
    size=n
    maxlength="n"
    />
    Note:size=n(set the size of the text box to be n char.)
